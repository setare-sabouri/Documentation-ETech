# Slider Component

Slider component is powered by Swiper.

```js filename="slider.js"  copy
const Slider = () => {
  return (
    <Swiper
      className="md**:w-70 xs**:w-60 lg:w-70 h-auto "
      modules={[Navigation, Scrollbar, A11y]}
      spaceBetween={50}
      slidesPerView={1}
      navigation
      scrollbar={{ draggable: true }}
      onSwiper={(swiper) => console.log(swiper)}
      onSlideChange={() => console.log('slide change')}
    >
      <SwiperSlide>
        <img src="./images/slider.png" />
      </SwiperSlide>
      <SwiperSlide>Slide 2</SwiperSlide>
      <SwiperSlide>Slide 3</SwiperSlide>
      <SwiperSlide>Slide 4</SwiperSlide>
    </Swiper>
  );
};
```

## The Swiper component is configured with various options

- **modules**: We include navigation, scrollbar, and accessibility modules.
- **spaceBetween**: It sets the space between slides.
- **slidesPerView**: It determines the number of slides visible at once (1 in this case).
- **navigation**: Enables navigation arrows.
- **scrollbar**: Configures a draggable scrollbar.
- **Event handlers** (onSwiper and onSlideChange) are also provided for customization.

  Inside the Swiper component, we have multiple SwiperSlide components, each containing content for a slide.
